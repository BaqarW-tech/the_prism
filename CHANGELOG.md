# Changelog

All notable changes to **The Prism** will be documented in this file.

## [1.0.0] - 2026-04-02

### Added
- Initial release of The Prism.
- News analysis engine with Liberal, Conservative, and Centrist lenses.
- Factual summary generation.
- Perspective comparison feature.
- Responsive UI with Tailwind CSS and Framer Motion.
- PDF export capability.
- GitHub documentation (README, CONTRIBUTING, LICENSE, CHANGELOG).
- Sample CSV data files.
